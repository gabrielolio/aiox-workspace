## O Serviço

O KING é um sistema de assistência para criadores de conteúdo automotivo. Não é um software que o cliente instala — é um serviço gerenciado que funciona pelo WhatsApp do criador.

**O que entregamos:**
- Acesso ao KING via WhatsApp (número dedicado)
- Setup personalizado pelas marcas do criador
- Onboarding + treinamento (1 sessão de 30 min)
- Suporte mensal + atualizações do sistema

## Diferenciais Competitivos

### Por que não é só um ChatGPT?

O KING é treinado especificamente para o mercado automotivo brasileiro. Ele conhece:
- Guidelines oficiais de Porsche, GWM e outras marcas
- Vocabulário automotivo brasileiro
- Calendário de produção de concessionárias
- Formato de legenda para cada plataforma (Instagram, YouTube, CapCut)

Um ChatGPT genérico pode inventar regras de guidelines. O KING nunca inventa — consulta a fonte oficial.

### Por que não é um software de legenda?

Softwares de legenda (Kapwing, Submagic, etc.) transcrevem e formatam. O KING vai além:
- Entende qual marca está no vídeo e aplica o template correto
- Sabe as restrições de cada marca antes mesmo de você perguntar
- Está conectado com o fluxo de briefing semanal
- Funciona pelo WhatsApp — zero curva de aprendizado

## Processo de Onboarding

### Semana 1 — Setup
- Conversa de levantamento (30 min): quais marcas, volume de vídeos, fluxo atual
- Configuração do KING para as marcas do criador
- Carregamento das guidelines (YAML por marca)
- Configuração do número WhatsApp dedicado

### Semana 2 — Piloto
- Criador usa em produção real com suporte próximo
- Ajustes de template conforme feedback
- Calibragem do Briefer para o calendário real do criador

### Semana 3+ — Autônomo
- Uso independente
- Check-in quinzenal (15 min)
- Atualizações de guidelines conforme marcas lançam novidades
