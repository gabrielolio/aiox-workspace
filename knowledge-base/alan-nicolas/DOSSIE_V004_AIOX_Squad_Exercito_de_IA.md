## PONTOS TÉCNICOS PRINCIPAIS

### 1. AIOX — A Renomeação Oficial

O framework antes chamado AIOS agora é **AIOX** (AI Orchestration eXtended ou AI-Orchestrated eXecution — nomenclatura exata não confirmada).

Mudanças técnicas confirmadas:
```bash
# Antes (AIOS)
npx aios-core init meu-projeto
@agente *comando

# Agora (AIOX)
npx aiox-core init meu-projeto
@agente *comando  # Claude Code (mantém @)
$agente *comando  # Codex 5.3 (usa $)
```

### 2. O "Exército de IA" — Arquitetura de Múltiplos Squads

A proposta do V004 é a evolução natural do V002:

