**Status:** ATIVO — Foco principal

## Sobre

Projeto de gestao de conteudo e identidade visual para a Fialho Motors, garagem de seminovos multimarcas com foco GWM em Campo Grande/MS.

Este e um projeto **independente do KING**. Opera 100% manual com templates reutilizaveis. As automacoes do KING podem ser aplicadas futuramente se o volume justificar.
